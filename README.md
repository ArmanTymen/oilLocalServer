# OilGasProject Local Server

Mock-сервер для локальной разработки фронтенда нефтегазового приложения.
Эмулирует данные по добыче и бурению, отправляет обновления через WebSocket.

## Стек

- Node.js + TypeScript
- Express
- Socket.IO

## Установка и запуск

```bash
npm install

# Запуск сервера в режиме разработки
npm run start

Сервер стартует на http://localhost:3001

REST API
GET /fields
Возвращает иерархический список месторождений, кустов и скважин с текущими показателями (дебит, давление, температура, и т.д.).

GET /drilling
Возвращает список буровых скважин с параметрами бурения, координатами забоя и историей.

WebSocket события
Сервер автоматически рассылает обновления всем подключённым клиентам:

fields:update — раз в 5 секунд, изменённые данные по дебиту и давлению для части скважин.

drilling:update — раз в 3 секунды, обновлённые параметры активных буровых скважин (глубина, координаты, ROP, давление и т.д.).

Форматы данных
Структуры описаны в файлах предметных областей:

Данные по добыче: src/data/production.data.ts

Данные по бурению: src/data/drilling.data.ts

Клиент получает только публичные поля (без коэффициентов дрейфа).
